The app is a calculator that does adding, subtracting, multiplcation, and division.
The app first asks the user for addition subttaction, multiplcation, division, or to quit.
Once one of them is selected the app asks for two numbers and does the request that the user asked for.
the app will output the answer and goes back to asking a operation.
The app will also save previous answers which can be used in the next operation.
